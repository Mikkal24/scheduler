About Task Scheduler

The Task Scheduler is a very simple application that allows users to run shell commands (aka tasks) at intervals using either milliseconds, CRON formatting or startup triggers. The Task Scheduler has a suite of features to make that simple objective a seemless experience for the end user relative to current market competitors.

The Task Scheduler aims at taking on the Windows Task Scheduler and Unix/Linux Crontab by providing a better user experience on all platforms including macOs.

The Task Scheduler is made by me, Michael Sorensen (sw0rdensen)



Task Scheduler Goals

The Task Scheduler has a long road ahead of it but, it will stay true to the following objectives:

  1. A better user interface
  2. Better security
  3. Support multiple operating systems (windows, mac, linux)
  4. Remain free and open source
